# Ethics and Limits

## EN

This project is a protocol research artifact. It must not be used to:
- Facilitate illegal activity
- Circumvent labor protections
- Pressure or manipulate workers
- Assign unsafe, coercive, or privacy-invasive tasks

### Human-first safeguards
- Informed consent before acceptance
- Clear right to refuse at any time before commitment
- Transparent compensation and cancellation policy
- Explicit no-retaliation clause

### Legal note
Jurisdiction-specific laws differ. This repository provides protocol design ideas, not legal conclusions. Always involve qualified counsel for production deployment.

### Governance note
If this protocol is implemented in a real service, operators should maintain:
- versioned policy packs,
- auditable decision logs,
- documented human override procedures.

## KR

본 저장소는 프로토콜 연구 산출물이며, 다음 목적으로 사용해서는 안 됩니다.
- 불법행위 조장
- 노동 보호장치 우회
- 수행자에 대한 압박/조작
- 안전 위협·강압·사생활 침해 업무 배정

### 인간 우선 보호원칙
- 수락 전 충분한 고지와 동의
- 확정 전 단계에서 언제든 거부할 권리
- 보상 및 취소 규칙의 투명한 고지
- 보복 금지 조항의 명시

### 법률 유의사항
국가·지역별 법률은 상이합니다. 본 저장소는 법률 결론이 아닌 프로토콜 설계 아이디어를 제공합니다. 실제 서비스 배포 전에는 반드시 자격 있는 법률 검토를 수행해야 합니다.

### 거버넌스 유의사항
이 프로토콜을 실제 서비스에 적용할 경우, 운영 주체는 최소한 다음을 유지해야 합니다.
- 버전 관리되는 정책 팩,
- 감사 가능한 의사결정 로그,
- 문서화된 인간 개입(override) 절차.
